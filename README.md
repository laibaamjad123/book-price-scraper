# 📚 Book Price Scraper

A Python web scraper that extracts book titles and prices 
from books.toscrape.com and saves them to a clean CSV file.

## 🛠️ Tools Used
- Python
- BeautifulSoup
- Requests

## 📊 Output
Clean CSV file with:
- Full book titles
- Prices in £

## ▶️ How to Run
pip install requests beautifulsoup4
python book_price_scraper.py
